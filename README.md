# Документация по умному матрасу

## Техническое обеспечение

Данная документация описывает работу комплекса «Умный матрас».  
Устройство представляет собой покрытие с умными датчиками давления и систему управления на базе операционной системы Linux, установленной на Orange Pi.

В комплекте с устройством поставляется экран Zeuslap, оснащённый следующими портами:
- USB Type-C
- mini-HDMI
- разъём для зарядки

## Программное обеспечение

Приложение настроено на автозапуск и запускается автоматически через некоторое время после включения устройства.

Для внесения изменений, тестирования и других нужд можно получить доступ к системе и запустить программу вручную.

### Запуск приложения вручную

1. Откройте терминал в папке с проектом, расположенной по пути:  
   `/home/orangepi/SensorsCol`

2. Активируйте виртуальное окружение командой:
   ```bash
   source venv/bin/activate
   ```

3. Запустите скрипт:
   ```bash
   python3 app2_rev_04_02_26_screens.py
   ```

### Настройка автозапуска

Для изменения параметров автозапуска необходимо отредактировать конфигурационный файл службы systemd.

1. Откройте файл службы с помощью редактора:
   ```bash
   sudo nano /etc/systemd/system/sensorscol.service
   ```

2. Внесите необходимые изменения.

3. Сохраните изменения, нажав комбинацию клавиш:  
   `Ctrl + S`

4. Выйдите из редактора обратно в терминал, нажав:  
   `Ctrl + X`

5. Перезагрузите конфигурацию systemd:
   ```bash
   sudo systemctl daemon-reload
   ```

6. Перезапустите сервис:
   ```bash
   sudo systemctl restart sensorscol.service
   ```
```
